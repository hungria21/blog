# Introdução ao Git e GitHub: Guia Completo para Iniciantes

O **Git** é uma ferramenta essencial para qualquer desenvolvedor. Neste post, vou explicar os conceitos básicos e mostrar como usar o Git e GitHub de forma prática.

## O que é Git?

Git é um sistema de controle de versão distribuído que permite:

- Rastrear mudanças no código
- Trabalhar em equipe sem conflitos
- Manter histórico completo do projeto
- Criar branches para diferentes funcionalidades
- Reverter mudanças quando necessário

## Instalação

### Windows
```bash
# Baixe o Git para Windows em: https://git-scm.com/download/win
```

### Mac
```bash
# Usando Homebrew
brew install git
```

### Linux (Ubuntu/Debian)
```bash
sudo apt update
sudo apt install git
```

## Configuração Inicial

Após instalar o Git, configure suas informações:

```bash
git config --global user.name "Seu Nome"
git config --global user.email "seu-email@example.com"
```

## Comandos Essenciais

### 1. Inicializar um Repositório

```bash
# Criar um novo repositório
git init

# Clonar um repositório existente
git clone https://github.com/usuario/repositorio.git
```

### 2. Verificar Status

```bash
# Ver status dos arquivos
git status

# Ver histórico de commits
git log
```

### 3. Adicionar e Commitar

```bash
# Adicionar arquivos ao staging
git add arquivo.txt
git add .  # Adicionar todos os arquivos

# Fazer commit
git commit -m "Mensagem do commit"
```

### 4. Trabalhando com Branches

```bash
# Criar uma nova branch
git branch nova-funcionalidade

# Mudar para a branch
git checkout nova-funcionalidade

# Criar e mudar para branch (comando combinado)
git checkout -b nova-funcionalidade

# Listar branches
git branch
```

### 5. Merge e Push

```bash
# Fazer merge de uma branch
git checkout main
git merge nova-funcionalidade

# Enviar para repositório remoto
git push origin main
```

## GitHub: Hospedando seus Projetos

O **GitHub** é uma plataforma que hospeda repositórios Git na nuvem.

### Conectando com GitHub

```bash
# Adicionar repositório remoto
git remote add origin https://github.com/usuario/repositorio.git

# Enviar código para GitHub
git push -u origin main
```

### Fluxo de Trabalho Típico

1. **Clone** o repositório
2. **Crie** uma branch para sua feature
3. **Faça** suas alterações
4. **Commit** suas mudanças
5. **Push** para o GitHub
6. **Abra** um Pull Request

## Exemplo Prático

Vamos criar um projeto simples:

```bash
# 1. Criar pasta do projeto
mkdir meu-projeto
cd meu-projeto

# 2. Inicializar Git
git init

# 3. Criar arquivo
echo "# Meu Projeto" > README.md

# 4. Adicionar arquivo
git add README.md

# 5. Fazer primeiro commit
git commit -m "Primeiro commit"

# 6. Conectar com GitHub
git remote add origin https://github.com/usuario/meu-projeto.git

# 7. Enviar para GitHub
git push -u origin main
```

## Dicas Importantes

### Boas Práticas para Commits

- Use mensagens descritivas
- Faça commits pequenos e frequentes
- Siga um padrão de mensagens

```bash
# Bons exemplos
git commit -m "Adiciona função de login"
git commit -m "Corrige bug na validação de email"
git commit -m "Atualiza documentação da API"
```

### Ignorando Arquivos

Crie um arquivo `.gitignore`:

```bash
# Arquivos do sistema
.DS_Store
Thumbs.db

# Dependências
node_modules/
vendor/

# Arquivos de build
dist/
build/

# Arquivos de configuração
.env
config.local.js
```

## Comandos Úteis Extras

```bash
# Desfazer último commit (mantém alterações)
git reset --soft HEAD~1

# Desfazer alterações em arquivo
git checkout -- arquivo.txt

# Ver diferenças
git diff

# Atualizar repositório local
git pull origin main

# Criar tag
git tag v1.0.0
git push origin v1.0.0
```

## Conclusão

O Git e GitHub são ferramentas fundamentais para qualquer desenvolvedor. Com a prática, esses comandos se tornarão naturais no seu fluxo de trabalho.

**Próximos passos:**
- Pratique com projetos pessoais
- Aprenda sobre Git Flow
- Explore recursos avançados do GitHub
- Contribua para projetos open source

**Lembre-se**: A melhor forma de aprender Git é praticando!

# Guia Rápido: Git Básico

## Configuração Inicial
```bash
git config --global user.name "Seu Nome"
git config --global user.email "email@exemplo.com"
```

## Comandos Essenciais

### Inicializar/Clonar
```bash
git init                    # Inicializar repositório
git clone <url>             # Clonar repositório
```

### Status e Informações
```bash
git status                  # Ver status
git log                     # Ver histórico
git diff                    # Ver diferenças
```

### Adicionar e Commitar
```bash
git add <arquivo>           # Adicionar arquivo
git add .                   # Adicionar todos
git commit -m "mensagem"    # Fazer commit
```

### Branches
```bash
git branch                  # Listar branches
git branch <nome>           # Criar branch
git checkout <nome>         # Mudar branch
git checkout -b <nome>      # Criar e mudar
git merge <nome>            # Fazer merge
```

### Remoto
```bash
git remote add origin <url> # Adicionar remoto
git push origin main        # Enviar para remoto
git pull origin main        # Baixar do remoto
```

## Fluxo Básico
1. `git add .`
2. `git commit -m "mensagem"`
3. `git push origin main`

# Guia Rápido: JavaScript Essencial

## Variáveis
```javascript
// Declaração
let nome = "João";
const idade = 30;
var cidade = "São Paulo"; // Evite usar

// Tipos
let numero = 42;
let texto = "Hello";
let booleano = true;
let array = [1, 2, 3];
let objeto = { nome: "João", idade: 30 };
```

## Funções
```javascript
// Função tradicional
function somar(a, b) {
    return a + b;
}

// Arrow function
const multiplicar = (a, b) => a * b;

// Função anônima
const dividir = function(a, b) {
    return a / b;
};
```

## Arrays
```javascript
const frutas = ["maçã", "banana", "laranja"];

// Métodos úteis
frutas.push("uva");           // Adicionar
frutas.pop();                 // Remover último
frutas.forEach(f => console.log(f));  // Iterar
frutas.map(f => f.toUpperCase());     // Transformar
frutas.filter(f => f.length > 5);     // Filtrar
frutas.find(f => f === "banana");     // Encontrar
```

## Objetos
```javascript
const pessoa = {
    nome: "João",
    idade: 30,
    falar() {
        return `Olá, eu sou ${this.nome}`;
    }
};

// Acessar propriedades
console.log(pessoa.nome);
console.log(pessoa["idade"]);

// Destructuring
const { nome, idade } = pessoa;
```

## Condicionais
```javascript
// if/else
if (idade >= 18) {
    console.log("Maior de idade");
} else {
    console.log("Menor de idade");
}

// Ternário
const status = idade >= 18 ? "adulto" : "criança";

// Switch
switch (dia) {
    case 1:
        console.log("Segunda");
        break;
    case 2:
        console.log("Terça");
        break;
    default:
        console.log("Outro dia");
}
```

## Loops
```javascript
// for
for (let i = 0; i < 5; i++) {
    console.log(i);
}

// while
let i = 0;
while (i < 5) {
    console.log(i);
    i++;
}

// for...of (arrays)
for (const fruta of frutas) {
    console.log(fruta);
}

// for...in (objetos)
for (const key in pessoa) {
    console.log(key, pessoa[key]);
}
```

## Async/Await
```javascript
// Promise
const buscarDados = () => {
    return new Promise((resolve, reject) => {
        setTimeout(() => {
            resolve("Dados carregados");
        }, 1000);
    });
};

// Async/Await
async function carregarDados() {
    try {
        const dados = await buscarDados();
        console.log(dados);
    } catch (error) {
        console.error(error);
    }
}
```

## DOM
```javascript
// Selecionar elementos
const elemento = document.getElementById("meuId");
const elementos = document.querySelectorAll(".minhaClasse");

// Modificar conteúdo
elemento.textContent = "Novo texto";
elemento.innerHTML = "<strong>HTML</strong>";

// Adicionar evento
elemento.addEventListener("click", function() {
    console.log("Clicado!");
});
```

## Dicas Importantes
- Use `const` por padrão, `let` quando precisar reatribuir
- Evite `var` - use `let` ou `const`
- Sempre use `===` ao invés de `==` para comparação
- Use arrow functions para callbacks
- Aprenda sobre closures e hoisting
- Pratique com projetos reais

## Exemplo Prático
```javascript
// Lista de tarefas simples
const tarefas = [];

function adicionarTarefa(texto) {
    const tarefa = {
        id: Date.now(),
        texto: texto,
        completa: false
    };
    tarefas.push(tarefa);
    return tarefa;
}

function completarTarefa(id) {
    const tarefa = tarefas.find(t => t.id === id);
    if (tarefa) {
        tarefa.completa = true;
    }
}

function listarTarefas() {
    return tarefas.filter(t => !t.completa);
}
```
