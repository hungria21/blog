# Fontes de Bots do Telegram

Este arquivo contém uma lista de sites de catálogos, repositórios de código aberto e bots internos para encontrar e estudar bots do Telegram.

## 1. Catálogos de Bots (Web)

Estes sites oferecem interfaces amigáveis para navegar por categorias, ver avaliações e descobrir novos bots.

*   **[Telegramic](https://telegramic.org/bots/)**: Um dos maiores diretórios comunitários de bots, canais e grupos.
*   **[Bots Archive](https://botsarchive.com/)**: Um arquivo detalhado com foco em utilidade, música e jogos.
*   **[Best of Telegram](https://bestoftelegram.com/bots/)**: Catálogo curado que lista os melhores bots em diversas categorias.
*   **[Lyzem](https://lyzem.com/)**: Um motor de busca específico para conteúdo do Telegram, incluindo bots.
*   **[AirDroid - 20 Best Telegram Bot Finders](https://www.airdroid.com/ai-insights/telegram-bot-finder/)**: Uma lista de referência que agrega vários buscadores de bots.

## 2. Repositórios de Código Aberto (Fontes)

Repositórios no GitHub que listam bots open source ou servem como agregadores de projetos para desenvolvedores.

*   **[danyspin97/TelegramBotsList](https://github.com/danyspin97/TelegramBotsList)**: Uma lista abrangente de bots do Telegram com código aberto hospedado no GitHub, organizados por linguagem de programação.
*   **[TechGujarati/TelegramBots](https://github.com/TechGujarati/TelegramBots)**: Uma lista de bots úteis com links diretos e descrições.
*   **[Webdifferential492/Telegram-Bot-List](https://github.com/Webdifferential492/Telegram-Bot-List)**: Repositório que foca em diversos tipos de bots, desde mensagens automáticas até ferramentas de gerenciamento.
*   **[TelegramBots (Organização)](https://github.com/orgs/TelegramBots/repositories)**: Repositórios oficiais de frameworks e bibliotecas populares para criação de bots (ex: Telegram.Bot para .NET).
*   **[python-telegram-bot](https://github.com/python-telegram-bot/python-telegram-bot)**: Fonte de uma das bibliotecas mais usadas, excelente para estudar a implementação de bots em Python.

## 3. Bots e Canais de Busca (Internos do Telegram)

Você pode encontrar esses bots e canais diretamente na barra de busca do aplicativo Telegram.

*   **@BotFather**: O bot oficial do Telegram para criação e gerenciamento de bots.
*   **@tlgrmcbot**: Bot oficial do site Telegramic para busca interna.
*   **@BotList**: Um dos canais mais antigos e conhecidos para divulgação de novos bots.
*   **@StoreBot**: O clássico bot de "loja" para descobrir e avaliar bots.
*   **@BotsArchive**: Canal oficial que envia atualizações e novos bots adicionados ao site BotsArchive.
