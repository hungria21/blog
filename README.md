# Kodi Web Remote Projects

Este repositório contém duas versões de uma interface remota para o Kodi, inspirada no Chorus2:

## 1. Extensão para Google Chrome (`kodi_extension/`)
Uma extensão que permite controlar o Kodi diretamente da barra de ferramentas do navegador.
- **Instalação:**
  1. Vá em `chrome://extensions/`
  2. Ative o "Modo do desenvolvedor"
  3. Clique em "Carregar sem compactação" e selecione a pasta `kodi_extension`.

## 2. Addon de Interface Web para Kodi (`webinterface.julesremote.zip`)
Uma interface que pode ser instalada diretamente no seu sistema Kodi.
- **Instalação:**
  1. Baixe o arquivo `webinterface.julesremote.zip`.
  2. No Kodi, vá em `Configurações > Addons > Instalar a partir de um arquivo zip`.
  3. Após instalar, vá em `Configurações > Serviços > Controle > Interface Web` e selecione "Jules Remote".
  4. Acesse via navegador em `http://IP_DO_KODI:PORTA`.
