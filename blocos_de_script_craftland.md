# Lista de Blocos de Script do Free Fire Craftland (Oficina de Criação)

## Status da Solicitação

Após uma pesquisa aprofundada na documentação oficial do Garena Free Fire, em wikis de comunidades e outros recursos online, foi constatado que **não existe uma lista de texto pública, completa e exata de todos os blocos de script da Oficina de Criação.**

A documentação oficial fornece uma visão geral das **categorias** de blocos, mas não detalha cada bloco individualmente. A lista completa e interativa de blocos parece estar disponível **exclusivamente na interface do editor do Craftland, dentro do próprio jogo Free Fire.**

## Resumo das Categorias de Blocos

Abaixo está um resumo das categorias de blocos disponíveis, conforme a documentação oficial. Isso pode ajudar a entender a estrutura e a funcionalidade geral do sistema de scripts:

### 1. Evento (Event)
Blocos que iniciam uma sequência de ações quando algo acontece no jogo (ex: `Quando um jogador entra no jogo`, `No início da rodada`).

### 2. Condições (Conditions)
Blocos que controlam o fluxo da lógica com base em certas condições (ex: `Se / Senão`, `Verificar pontuação`).

### 3. Comportamento (Behavior)
Blocos que executam ações diretas no jogo (ex: `Mover um objeto`, `Alterar a vida de um jogador`).

### 4. Módulos (Modules)
Ações específicas para certos módulos ou modos de jogo do Craftland.

### 5. Dados (Data)
Blocos para manipular dados como números e textos (ex: `Operações matemáticas`, `Obter posição do jogador`).

### 6. Variáveis (Variables)
Para armazenar e reutilizar informações (ex: `Definir variável`, `Obter variável`).

### 7. Função (Function)
Para criar blocos de código reutilizáveis (funções customizadas).

### 8. Chamada Externa (External call)
Para chamar funções de outros scripts.

---
*Nota: A única fonte para uma lista exata e completa de todos os blocos e seus parâmetros é a interface do editor da Oficina de Criação no Free Fire.*
